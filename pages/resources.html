Using the Permit Checklist System

This permit checklist system is designed to help construction project teams track and manage all required permits, approvals, and inspections throughout the project lifecycle. Follow the steps below to get the most out of this tool:

  1. Navigate to the Interactive Checklist page
  2. Enter your project information in the Project Information card
  3. Click "Save Project Info" to store your project details
  4. Review all permit categories to identify which ones apply to your project
  5. For each applicable permit item, assign responsibility and update status

For each permit item in the checklist:

  1. Click the "Edit" button to open the edit modal
  2. Assign responsibility to a team member or organization
  3. Set the current status:
    • Pending - Not yet started
    • In Progress - Application submitted or under review
    • Approved - Permit obtained or requirement fulfilled
    • Not Applicable - Item does not apply to your project
  4. Add cost estimates if available
  5. Add notes about specific requirements, deadlines, or contacts
  6. Click "Save Changes" to update the item

The Project Dashboard provides visual tracking of your permit progress:

  • View overall project progress percentage
  • See status breakdown pie chart
  • Track progress by permit category
  • Monitor pending items that need attention
  • View recent activity and updates
  • Use the "Refresh Dashboard" button to update with the latest data

To generate a report of your project's permit status:

  1. Go to the Interactive Checklist
  2. Click the "Generate Report" button in the Checklist Progress card
  3. A JSON file containing all permit data will be downloaded
  4. This report can be used for:
    • Team meetings and updates
    • Backing up your project data
    • Sharing status with stakeholders
    • Record-keeping and documentation

The current version does not support adding custom permit items. However, you can use the Notes field to track additional information for existing items. A future update will include functionality to add, edit, and customize permit items.

The current version stores data locally in your browser, which means each user would have their own copy of the data. For team collaboration, consider implementing a cloud-based version of this application or regularly sharing the generated reports with team members.

To clear your project data and start a new project, you can clear your browser's local storage for this site. In most browsers, you can do this by:
  1. Opening the browser's Developer Tools (F12 or right-click and select "Inspect")
  2. Going to the "Application" or "Storage" tab
  3. Finding "Local Storage" in the sidebar
  4. Right-clicking on the site URL and selecting "Clear"
After clearing the storage, refresh the page and you'll start with a clean slate.

Currently, the system exports data in JSON format. For PDF or Excel formats, we recommend opening the dashboard or checklist page and using your browser's print function (Ctrl+P or Cmd+P) to generate a PDF. Alternatively, you can import the JSON file into a spreadsheet program for further processing.
